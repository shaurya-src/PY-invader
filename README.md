# PY-invader

_Classic Space Invader Arcade using Python._

### Library used : 
        1. pygame
        2. random
        3. math

### Game Evironment : 
        
        You control a Spaceship and your goal is to defend it from the invading aliens.
        As long as you can keep them far from your territory, game goes on.
    
### Controls :

        1. LEFT : Steer left
        2. RIGHT : Steer right
        3. UP : Fire
