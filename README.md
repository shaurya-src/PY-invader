# PY-invader

_Classic Space Invader Arcade using Python.

### * Library used : 'pygame'

### * Game Evironment : 
        
        You control a Spaceship and your goal is to defend it from the invading aliens.
        As long as you can keep them far from your territory, game goes on.
    
### * Controls :

        LEFT : Steer left
        RIGHT : Steer right
        UP : Fire
